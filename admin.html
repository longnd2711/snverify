Hệ thống HEM EMM - Cập nhật dữ liệu hàng loạt từ file Excel

Lưu ý cấu trúc file Excel:

  • File tải lên phải là .xlsx hoặc .csv.
  • Dữ liệu phải có chính xác 20 cột theo đúng thứ tự của Sheet "Data".
  • Dòng đầu tiên (dòng 1) của file Excel sẽ được coi là Tiêu đề cột và sẽ bị bỏ qua khi import.
  • Chỉ import các dòng có chứa số Serial (cột A không được để trống).
